投资池开放日维护

维护投资池开放区间（begin_date ~ end_date）。仅当投资池开启 open_day_adjust 时，证券池 / 禁投 / CRMW / 批量调库才会按本表校验调库当日是否处于开放窗口。

业务流程总览

从页面操作到数据库读写的主要路径。

1. 加载池树读取未删除投资池树，供筛选与表单选择叶子池。
2. 分页查询按池、日期区间、描述筛选开放日配置。
3. 新增区间校验叶子池与起止日，写入 ip_pool_open_day。
4. 修改区间按主键更新池、起止日与描述。
5. 逻辑删除is_deleted=1，调库校验不再命中该区间。

查询与列表展示

列表关联投资池展示全称与池级开放日开关快照。

POST /api/v1/poolOpenDay/queryPoolOpenDayPage · 投资池树同 investmentPool 查询

筛选条件

  • 投资池：树单选，目录/根节点（含子节点）不可选，与表单选择规则一致。
  • 日期区间：按 begin_date / end_date 与查询上下限交叉过滤（前端 daterange）。
  • 描述：模糊匹配 description。
  • 仅 is_deleted=0；排序 begin_date DESC, id DESC。

列表字段与前端状态

  • 返回池全称 poolFullName、池级 openDayAdjust（是否启用开放日校验）。
  • 「区间状态」前端按当天计算：未开始 / 开放中 / 已结束（不入库）。
  • 操作列固定右侧：编辑、删除；数据加载后 doLayout 防固定列错位。

新增、修改与删除

写操作均校验参数与叶子池；修改/删除按主键定位未删除记录。

POST /api/v1/poolOpenDay/addPoolOpenDay · editPoolOpenDay · deletePoolOpenDay

新增 / 修改

  • 必填：poolId、beginDate、endDate；endDate 不得早于 beginDate。
  • 投资池须存在且未删除；有子池的目录节点禁止配置。
  • description 可空；前端 maxlength=100（库字段 VARCHAR(1000)）。
  • 新增后回查详情；修改更新 updt_time。

删除

  • 逻辑删除：is_deleted=1，写 updt_time。
  • 已删除记录不可再改；调库开放日命中仅统计 is_deleted=0 区间。

调库校验关联

本页只维护区间行；池级开关在「投资池维护」的 open_day_adjust。

何时生效

  • 目标池 open_day_adjust=1 时，证券池 / 禁投 / CRMW / 批量等调入与调出均校验开放日。
  • 命中条件：begin_date ≤ 当天 ≤ end_date 且 is_deleted=0（日期口径与实现一致）。
  • open_day_adjust=0 时本表不参与校验；未命中任何区间时由各调库服务按「不在开放日」处理。

与其他约束的关系

  • 与 lock_flag（池锁定）、frozen_period_in（入池冻结期）相互独立。
  • 同一池可配置多段历史/未来区间；调库只认「当日命中」的未删除行。
  • 不修改投资池树本身；叶子池与开关配置见投资池维护文档。

涉及表、用途与触发场景

读写口径包含页面初始化、主业务接口及调库校验读取。

表名 用途 读写 触发场景
ip_pool_open_day 投资池开放区间主表（pool_id + begin/end + description） 查询 新增 更新/逻辑删除 列表、新增、编辑、删除；调库开放日校验读取
ip_investment_pool 池树与 open_day_adjust 开关；列表联查全称 查询 树选择、详情联查、目录节点校验（子池数）
主表字段要点（ip_pool_open_day）
  • id：主键
  • pool_id：投资池 ID → ip_investment_pool.id
  • begin_date / end_date：开放区间（DATE，含边界）
  • description：描述，库 VARCHAR(1000)，前端录入限制 100
  • is_deleted：0 正常 / 1 已删除
  • crte_time / updt_time：创建 / 修改时间

规则说明

实现与页面口径摘要。

  • 业务页：pages/pool_open_day.html；接口前缀 /api/v1/poolOpenDay/。
  • 池级开关在投资池维护配置；本页不改 open_day_adjust。
  • 有子节点的目录/根池不可配置开放日，须选叶子池。
  • Schema 脚本：rrs_pool_init_schema.sql 中 ip_pool_open_day。

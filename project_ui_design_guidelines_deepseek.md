# 项目整体风格总结

本文档基于 `D:\my-project-all\znty-rrs-slice` 下 6 个切图模块的根目录 `index.html` 分析整理，用于后续统一 `znty-rrs-ui` 前端页面视觉风格。

## 1. 整体视觉定位

整体风格为金融业务型后台界面：白底、蓝色品牌主色、灰蓝文字体系，强调表格密度、信息扫描效率和业务稳重感；局部使用金色体现品牌强调或重点信息。

关键词：

- 商务稳重
- 简洁扁平
- 信息密集
- 表格优先
- 蓝金品牌感

## 2. 色彩体系

### 2.3 Element UI 色彩处理

切图不是 Element UI 默认蓝色体系，未使用 `#409EFF`、`#67C23A`、`#E6A23C`、`#F56C6C` 等默认色。后续改造 Element UI 页面时，应通过 CSS 覆盖为上述自定义品牌色。

建议：

- `el-button--primary` 主色统一为 `#2743A4`。
- 链接、文字按钮、选中态统一为 `#2743A4`。
- 表格边框统一为 `#E5E5E5`。
- 表头背景统一为 `#F7F8FA`。

## 3. 字体系统

### 3.1 字体族

主字体使用：

```css
font-family: "PingFang SC", "Microsoft YaHei", Arial, sans-serif;
```

切图中主要出现 `PingFang SC`、`PingFangSC-Regular`、`PingFangSC-Semibold`、`PingFangSC-Medium`。前端实现时不需要精确区分字体文件，用系统字体族即可。

### 3.2 字号层级

| 层级 | 字号 | 字重 | 行高 | 使用场景 |
|------|------|------|------|----------|
| 大数字 | `24px` / `28px` | 600 | `32px` / `34px` | 指标数字、统计卡重点值 |
| 页面标题 | `20px` / `22px` | 600 | `28px` / `30px` | 页面主标题、重要模块标题 |
| 区域标题 | `16px` | 600 | `24px` | 卡片标题、分类标题、表单区块标题 |
| 常规正文 | `14px` | 400 | `22px` | 按钮、表单、普通正文、图表图例 |
| 表格正文 | `13px` | 400 | `21px` / `22px` | 表格单元格 |
| 表格表头 | `13px` | 600 | `21px` / `22px` | 表头字段 |
| 辅助说明 | `12px` | 400 | `20px` | 标签、小型切换项、弱提示 |

### 3.3 字重规律

- 普通正文：`400`
- 标题、表头：`600`
- 少量强调：`500`
- 大型展示数字可按场景使用 `600`，不建议大面积使用 `700` 以上。

## 4. 间距与尺寸规范

### 4.1 间距

整体符合 4px 基准网格，常用间距为：

| 用途 | 建议值 |
|------|--------|
| 紧凑内边距 | `4px` / `8px` |
| 表单项横向间距 | `8px` / `12px` |
| 模块内间距 | `16px` |
| 区块间距 | `20px` / `24px` |
| 页面外边距 | `16px` / `24px` |
| 大区块间距 | `32px` |

后续页面建议优先使用 `4 / 8 / 12 / 16 / 20 / 24 / 32`，减少 `15px`、`45px` 等切图坐标型特例。

### 4.2 控件尺寸

| 组件 | 建议尺寸 |
|------|----------|
| 默认按钮 | 高 `32px`，圆角 `4px`，左右内边距 `12px` / `16px` |
| 小按钮 | 高 `28px`，圆角 `4px` |
| 输入框 | 高 `32px`，圆角 `4px` |
| 选择框 | 高 `32px`，圆角 `4px` |
| 标签 | 高 `24px` / `32px`，圆角 `4px` |
| 表格行 | 高 `32px` 或按 Element UI 压缩到接近该密度 |
| 卡片/面板 | 圆角 `4px` / `8px` |
| 图表柱 | 顶部圆角 `2px` |

## 5. 组件样式规范

### 5.1 按钮

主按钮：

- 背景：`#2743A4`
- 文字：`#FFFFFF`
- 圆角：`4px`
- 字号：`14px`
- 高度：`32px`

次按钮：

- 背景：`#FFFFFF`
- 边框：`#E5E5E5` 或品牌蓝线框
- 文字：`#3F485A`，强调操作可用 `#2743A4`

文字按钮：

- 默认：`#2743A4`
- 悬停/激活：可加深为 `#1F3683`

### 5.2 表格

表格是该项目最核心的信息承载组件。

- 表头背景：`#F7F8FA`
- 表头文字：`13px / 600 / #242933`
- 内容文字：`13px / 400 / #3F485A`
- 边框/分割线：`#E5E5E5`
- 单元格背景：`#FFFFFF`
- 行高：优先接近 `32px`
- 操作列文字按钮：品牌蓝 `#2743A4`

### 5.3 表单与输入框

- 输入框高度：`32px`
- 圆角：`4px`
- 边框：浅灰体系，建议 `#E5E5E5`
- 占位文字：辅助灰蓝，可用 `#9199BD`
- 已输入文字：`#3F485A`
- 表单标签：`14px / #3F485A`
- 错误状态：`#FF4542`

### 5.4 标签

可选标签：

- 默认：浅灰或浅蓝底，正文色 `#242933`
- 选中：`#2743A4` 底 + `#FFFFFF` 字
- 禁用：`#F6F6F6` 底 + 弱文字色

状态标签：

- 成功：`#06C687` / `#E7F9F3`
- 错误：`#FF4542` / `#FFEDEC`
- 警告：`#FFC802` / `#FFF4CC`
- 橙色：`#F5822E` / `#FDE6D5`

### 5.5 导航与菜单

- 菜单默认文字：`#3F485A`
- 选中态：`#2743A4`
- 选中背景：可用 `#E9ECF6` 或浅蓝透明底
- 图标尺寸主要为 `16px` / `24px`
- 顶部导航应保持白底、轻分割线、低装饰感

### 5.6 图表

- 主图形色：`#2743A4`
- 轴线/网格线：`#E5E5E5`
- 坐标轴文字：`#9199BD`
- 图例文字：`#9199BD`
- 强调色可使用金色 `#D1A777` 或状态色，但不要过多混用。

### 5.7 弹窗、气泡、面板

- 背景：`#FFFFFF`
- 圆角：`4px` / `8px`
- 阴影：轻量投影
- 标题：`16px / 600 / #242933`
- 内容：`14px / 400 / #3F485A`

## 6. 各模块差异点

| 模块 | 特征 |
|------|------|
| PC规范 | 组件覆盖最完整，是后续改造的主要参考源 |
| 内部排名总览 | 表格密度最高，状态色和排名色更丰富 |
| 内评计算 | 表单、输入、多选、单选、流程录入特征明显 |
| 实盘收益 | 图表占比更高，品牌蓝和金色强调更明显 |
| 研报检索 | 搜索、选择框、多选筛选、标签较突出 |
| 重点推荐 | 推荐卡片、标题、筛选项较多，出现珊瑚色和辅助绿 |

## 7. 前端落地建议

### 7.1 优先统一 CSS 变量

建议在每个页面样式中逐步收敛到同一套变量命名：

```css
:root {
  --xy-primary: #2743A4;
  --xy-primary-active: #1F3683;
  --xy-text-title: #242933;
  --xy-text-body: #3F485A;
  --xy-text-secondary: #9199BD;
  --xy-border: #E5E5E5;
  --xy-bg-page: #F7F8FA;
  --xy-bg-white: #FFFFFF;
  --xy-bg-primary-light: #E9ECF6;
  --xy-brand-gold: #D1A777;
  --xy-success: #06C687;
  --xy-danger: #FF4542;
  --xy-warning: #FFC802;
}
```

### 7.2 页面改造顺序

1. 先统一全局字体、页面背景、正文颜色。
2. 再统一 Element UI 主按钮、文字按钮、输入框、选择框。
3. 然后统一表格表头、边框、字号、行高。
4. 最后处理各页面独有的卡片、标签、图表和状态色。

### 7.3 风格一致性原则

- 少用大面积渐变和强阴影。
- 少用纯黑文字，优先使用灰蓝文字体系。
- 页面主体以白底和浅灰分割为主。
- 业务页面优先保证信息密度和扫描效率。
- 新增状态色必须先归入统一 token，不要在页面内随意新增孤立色值。
